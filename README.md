## 📚 Программа

### Блок 1 — Создание React-приложений

Элементы и компоненты, хуки (`useState`, `useEffect`, `useReducer`, `useRef`,
`useCallback`, `useMemo`), кастомные хуки, базовая теория рендеринга, формы,
CSS Modules, порталы и модальные окна.

### Блок 2 — Работа с данными

Организация данных, Context API, state managers, Redux, Redux Toolkit,
асинхронные запросы через thunk, RTK Query.

### Блок 3 — SPA и роутинг

React Router: статические, динамические и вложенные роуты, навигация,
редиректы, работа с параметрами пути и query-параметрами.

### Блок 4 — Продвинутый React

Логика рендеринга и Fiber «под капотом», оптимизация рендеринга, паттерны,
Suspense, `lazy` / динамические импорты, оптимизация работы с данными.

## 🛠 Технический стек

Проект инициализирован через **Vite** с шаблоном `react` (HMR + ESLint).

Официальные плагины React для Vite:

- [`@vitejs/plugin-react`](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) — использует [Oxc](https://oxc.rs)
- [`@vitejs/plugin-react-swc`](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) — использует [SWC](https://swc.rs/)

### Запуск проекта

```bash
npm install      # установка зависимостей
npm run dev      # дев-сервер с HMR
npm run build    # production-сборка
npm run preview  # предпросмотр сборки
```
